Services

UI / UX Design

Desainer UI UX adalah seorang profesional yang bertanggung jawab merancang antarmuka pengguna (UI) dan pengalaman pengguna (UX) suatu produk atau aplikasi.

Web Development

Web development, atau pengembangan web, adalah proses pembuatan dan pemeliharaan situs web atau aplikasi web. Proses ini mencakup berbagai aspek, mulai dari desain dan pemrograman hingga manajemen konten dan pemeliharaan.

Front-end Development

Front-end Development adalah berkaitan dengan apa yang dilihat oleh pengguna di situs web. Front-end developers menggunakan bahasa seperti HTML, CSS, dan JavaScript untuk membangun antarmuka pengguna yang interaktif dan responsif.
